# StudentHub

## Overview

StudentHub is a student-centric productivity platform being developed over a 6-week bootcamp. The project aims to create a unified digital workspace that helps students organize academic activities, track progress, manage resources, and improve day-to-day productivity.

The application is being developed incrementally, with each week introducing new features and technologies that build upon the previous stage.

---

## Project Goal

To create a centralized platform that helps students manage academic activities, productivity, and essential campus-related resources in one place.

---

# Development Roadmap

This project will be developed incrementally over a 6-week bootcamp, with each stage building upon the previous one.

| Week   | Planned Milestone                             |
| ------ | --------------------------------------------- |
| Week 1 | Live, responsive landing page                 |
| Week 2 | Interactive dashboard with persistent data    |
| Week 3 | React-powered frontend with routing           |
| Week 4 | Node.js + MongoDB backend with REST APIs      |
| Week 5 | JWT-based authentication and cloud deployment |
| Week 6 | Dockerization, CI/CD, and final live demo     |

---
